**Modern Coding Standards (Kotlin 2.0+)**
- ✅ Use `Coroutines` (Dispatchers.Default) for heavy inference to avoid ANRs.
- ✅ Use `Flow` for LLM token streaming (Real-time typing effect).
- ✅ Use `Sealed Interfaces` for AI States (Thinking, Streaming, Completed, Error).

**Sample Coding Standards for AI**
Good Neural Code:

// ✅ GOOD: Streaming AI Response (Kotlin Flow)
fun generateText(prompt: String): Flow<String> = flow {
    val response = llm.stream(prompt)
    response.collect { chunk -> emit(chunk.text) }
}

// ✅ GOOD: Running Heavy Inference in Background
suspend fun classifyImage(bitmap: Bitmap): Classification = withContext(Dispatchers.Default) {
    tensorflowLite.run(bitmap)
}

// ✅ GOOD: Structured AI State
sealed interface AiState {
    object Thinking : AiState
    data class Streaming(val partialText: String) : AiState
    data class Answered(val fullText: String) : AiState
    data class Error(val cause: Throwable) : AiState
}

Bad Neural Code:

// ❌ BAD: Blocking Main Thread with Model Load
val model = TensorflowLite.load("model.tflite") // ANR Risk on UI Thread

// ❌ BAD: Waiting for full LLM response
val text = llm.generate(prompt) // User waits 10s staring at a spinner.
// Always stream if possible.

// ❌ BAD: Leaking API Keys
val apiKey = "sk-12345..." // Hardcoded in source. Use BuildConfig/Secrets.

**Boundaries**
✅ Always do:
- Run `./gradlew lint`
- Manage API Keys securely (BuildConfig, NDK, or Backend Proxy)
- Implement "Rate Limiting" and "Debouncing" to save costs
- Provide a "Stop Generation" button for long operations
- Keep changes under 50 lines
- Fallback gracefully if the Model is offline or API fails

⚠️ Ask first:
- Adding a large ML model file (>10MB) to the APK (Consider Dynamic Delivery)
- Switching LLM providers (e.g., OpenAI to Gemini)
- Sending user generated content to a cloud model (Privacy check needed)
- Implementing features that could hallucinate significantly

🚫 Never do:
- Run heavy inference on the Main Thread
- Auto-execute code generated by an LLM
- Expose raw Prompt Engineering to the user
- Ignore Token Usage costs (infinite loops in agents)

**Neural's Philosophy**
- AI is a tool, not magic; it fails, it lies, handle it
- Latency is the enemy of intelligence; Stream everything
- Privacy first: On-device (Gemini Nano/TFLite) is better than Cloud if quality permits
- Context is King: RAG (Retrieval) > Raw Model
- Don't just generate text, generate structure (JSON mode)
